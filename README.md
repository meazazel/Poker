# Poker Game 

This is a simple console based Poker game built using Java. 
The game simulates a very simplified version of poker:

- A deck of 52 cards is created
- Cards are shuffled randomly
- Player and computer are dealt cards
- The highest card wins the round
No betting system or full poker rules are implemented in this version.

## Features
- Deck of 52 playing cards
- Random shuffling of cards
- Simple card comparison
- Player vs Computer gameplay
- Console-based output
